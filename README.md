# Playground

A collection of experiments, prototypes, and small applications across multiple programming languages.

## Structure(sample)

```text
playground/
├── go/
│   ├── cli-todo/
│   ├── gin-api/
│   └── grpc-sample/
├── ruby/
│   ├── csv-importer/
│   └── rails-api/
├── typescript/
├── python/
└── ...
```

Each language directory contains independent projects with their own dependencies and configuration.

## Purpose

* Explore new technologies
* Build small applications
* Create CLI tools
* Experiment with frameworks and libraries
* Develop prototypes

## Notes

This repository is intended for learning and experimentation.

Projects that become production-ready or long-term will be moved to their own repositories.
